# ARAdvertisementDemo
DEMO(for Mixed Reality System)

# 複合現実感の課題：AR広告
AR広告の世界観を掴むためのDEMOです．
このDEMOではジュークボックスが使用者の背後に現れ， 画面をタッチするとオブジェクトが生成され音が流れます．
新宝島が爆音で流れるので音量注意です．Objectから離れることで音は小さくなります．
